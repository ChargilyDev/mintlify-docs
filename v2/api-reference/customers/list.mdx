---
title: "List all customers"
description: "Returns a list of all your customers."
api: "GET /customers"
---

## Query parameters

<ParamField query="per_page" type="integer">
   A numeric value indicating the quantity of customers to be returned with the
   request, with a minimum value of 1 and a maximum value of 50.
</ParamField>

## Returns

A key-value array, containing a data property which is an array of customers, limited to a maximum of `per_page` value.
