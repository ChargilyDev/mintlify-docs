---
title: 'Retrieve a checkout'
description: "Retrieves all the information of an already-existing checkout by providing it's unique identifier (ID)."
api: 'GET /checkouts/{id}'
---

## Path parameters

<ParamField path="id" type="string">
  The ID of the checkout.
</ParamField>

## Returns

If a valid ID is provided, it returns a checkout object.
