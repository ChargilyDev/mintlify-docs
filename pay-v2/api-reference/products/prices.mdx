## Path parameters

<ParamField path="id" type="string">
   The ID of the product.
</ParamField>

## Query parameters

<ParamField query="per_page" type="integer">
   A numeric value indicating the quantity of prices to be returned with
   the request, with a minimum value of 1 and a maximum value of 50.
</ParamField>

## Returns

A key-value array, containing a data property which is an array of all the prices of the product, limited to a maximum of `per_page` value.
