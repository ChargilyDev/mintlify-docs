## Body parameters

<ParamField body="name" type="string" required>
The name of the product.
</ParamField>

<ParamField body="description" type="string">
The description of the product.
</ParamField>

<ParamField body="images" type="string[]">
Up to 8 URLs of images of the product.
</ParamField>

<ParamField body="metadata" type="key-value array">
A Set of key-value pairs that can be used to store additional information about the product.
</ParamField>

## Returns
